# Exam Score Calculator

## Description
Enter exam scores and calculate the average. The program shows a final grade: Excellent, Good, Pass, or Fail. Works for one score (v1) or multiple scores (v2).

# Features
- Input one or multiple scores
- Quit anytime with q
- Scores validated between 0 and 100
- Displays average and final grade

# Exam Score Calculator

## Description
This is a simple Python program that allows the user to enter multiple exam scores and calculates the average score. Based on the average, the program displays a final grade such as Excellent, Good, Pass, or Fail.

## Features
Accepts multiple exam scores from the user
Allows the user to quit input at any time by typing q
Validates inputs to ensure scores are between 0 and 100
Calculates and displays the average score
Shows a final grade based on the average result

## How It Works
The program repeatedly asks the user to enter exam scores. Each valid score is stored in a list. When the user finishes entering scores, the program calculates the average and determines the final grade using predefined score ranges.
Technologies Used
Python
Basic concepts such as loops, lists, conditional statements, and error handling

# Wall Builder

A simple Python program that prints a rectangular "wall" of a user-defined size and symbol. This project demonstrates basic Python concepts like loops, functions, input validation, and string manipulation.

## Features
- Prompts the user for the number of rows and columns (must be positive integers)
- Allows the user to choose a symbol to build the wall
- Validates user input and handles incorrect entries gracefully
- Prints a clean, rectangular wall in the console

## Example
Enter the number of rows: 3 
Enter the number of columns: 5 
Enter the symbol: *


## Purpose
This program is designed as a beginner-friendly Python project for learning input handling, loops, and basic function usage. Suitable for showcasing on GitHub and including in a portfolio for scholarships or university applications.

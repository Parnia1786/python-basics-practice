# Countdown Timer

A simple Python program that counts down from a user-defined number of seconds and shows the time in hh:mm:ss format.

## Features
- Input time in seconds
- Displays countdown every second
- Shows a message when time is up

## How to Use
- Make sure you have Python installed on your computer.
- Download the program file (CountdownTimer.py).
- Open a terminal (or command prompt) and go to the folder where the file is saved.
- Run the Python file.
- Enter a number representing seconds when prompted.
- The timer will start counting down automatically.
- When the countdown reaches zero, "TIME'S UP!" will be displayed.


## Example
- Enter the time in seconds: 5 
00:00:05 
00:00:04 
00:00:03 
00:00:02 
00:00:01 
TIME'S UP!
